Authors: Matt Williamson (mwilliamson@dawsdesign.com).

Description

This application is used to store and retrieve file chunks in an ErlFS cluster. When a client wants to retrieve a chunk of a file, it calls chunk_get/2. If it wants to store a chunk, it calls chunk_store/2.
